월드 와이드 웹, 그리고 프론트엔드와 백엔드

웹이 무엇이고 어떠한 특징이 있는지, 프론트엔드와 백엔드, 그리고 사용되는 언어들은 어떤 것들이 있는지에 대해서 알아봅니다.

2021.07.29

웹

현대인들은 살아가면서 수많은 웹사이트를 보게 됩니다. 검색을 위한 포털사이트, 어떠한 기업의 홈페이지, 혹은 다른 사람들과 소통을 할 때에도 웹을 접하게 되는데요, 이러한 웹은 아무리 멀리 떨어진 사람들과도 순식간에 소통을 할 수 있게 해주고 엄청난 양의 정보들도 금새 알아낼 수 있게 해줍니다. 우리들에게 이렇게 놀라운 혜택을 주는 웹에 대해서 지금부터 알아보도록 하겠습니다.

월드 와이드 웹

우리가 흔히 웹이라고 줄여서 말하는 월드 와이드 웹(World Wide Web)은 인터넷에 연결된 사용자들이 정보를 공유할 수 있는 공간을 말합니다. 흔히 인터넷과 혼용되어 사용되지만 보다 정확하게 이야기하자면 인터넷의 수많은 서비스들 중 가장 인기있는 서비스가 웹이라고 할 수 있습니다.

웹은 텍스트나 이미지, 소리 등의 정보들을 담은 문서들을 하이퍼텍스트(Hypertext) 방식으로 연결하여 제공합니다. 하이퍼텍스트란 한 문서를 다른 문서와 서로 연결시키는 방식을 의미하는데, 이때 다른 문서와 연결되어 이동할 수 있게 하는 링크를 하이퍼링크(Hyperlink)라고 합니다. 우리는 수많은 하이퍼링크들을 통해 웹에 존재하는 다양한 문서들을 읽고 받아들일 수 있으며 인터넷으로 연결된 많은 사람들과 소통이 가능합니다.

이러한 웹사이트를 만들기 위한 기술들을 크게 두가지로 분류하는데 바로 프론트엔드(Front-End) 와 백엔드(Back-End) 입니다.

프론트엔드(Front-End)

사용자가 웹사이트에 방문하게 되면 각 사이트의 특징에 맞게 디자인되어있는 페이지를 보게 됩니다. 이렇게 사용자가 직접 접하는 인터페이스를 프론트엔드(Front-End) 라고 부릅니다. 이러한 프론트엔드를 다루는 개발자를 프론트엔드 개발자라고 부르며 이들은 사용자가 알아보기 쉬운 인터페이스(UI)와 적절한 사용자경험(UX)을 만드는 데 초점을 두고 있습니다.

프론트엔드 분야에서 사용되는 언어는 주로 3가지가 있는데 HTML, CSS, JavaScript 입니다.

HTML은 Hyper Text Markup Language 의 약자로 웹페이지를 만들기 위한 마크업 언어입니다. 웹페이지의 커다란 구조를 만들 수 있으며 다른 페이지와의 링크를 만들 수 있습니다.

CSS는 HTML로 작성된 문서가 실제로 웹페이지에 어떻게 나타날 지를 지정하는 언어입니다. HTML이 웹페이지의 핵심적인 정보와 구조를 나타냈다면 CSS는 이들의 레이아웃과 스타일을 정의하여 보다 깔끔하고 알아보기 쉽도록 만들어줍니다.

JavaScript는 객체 기반의 스크립트 프로그래밍 언어입니다. 주로 웹브라우저 내에서 사용되며 보다 능동적인 웹페이지를 만들기 위해 사용됩니다. 주기적으로 페이지의 내용을 변화시키거나 사용자의 행동과 상호작용하여 여러가지 효과를 발생시킬 수 있습니다.

백엔드(Back-End)

프론트엔드가 사용자가 직접 마주하는 부분을 다루었다면 백엔드는 사용자가 보지 못하는 부분을 다룹니다. 백엔드 개발자는 웹사이트의 서버측을 개발하여 다양한 처리 프로세스와 데이더베이스 등을 관리하고 웹사이트가 잘 운영될 수 있도록 노력합니다.

백엔드 개발에는 PHP, Java, Python, 앞서 나왔었던 Javascript 등 다양한 언어들이 사용됩니다.

PHP는 웹을 위해 만들어진 프로그래밍 언어로 서버 측 개발에 있어서 높은 웹프로그래밍 생산성을 보여주며 편리하게 데이터베이스와 연동이 가능합니다.

Java는 한국 내 웹 백엔드에서 가장 높은 점유율을 보여줄 정도로 널리 사용되는 언어입니다. 관련 커뮤니티도 크게 활성화되어있으며 대규모 프로젝트에 적합합니다.

Python은 웹개발만을 위한 언어가 아닌 범용 언어로써 여러가지 시스템을 효율적으로 통합할 수 있게 해줍니다.

Node.js는 프론트엔드에서 등장하였던 Javascript의 문법을 사용하여 프론트엔드 개발자가 쉽게 익힐 수 있습니다. 처리 속도가 굉장히 뛰어나 많은 요청을 빠르게 처리할 수 있습니다.
